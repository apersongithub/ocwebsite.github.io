Setting

A setting holds value for the ui to change and the config to save/load

We will pretend that "setting" is a Setting


setting.type
The type of that setting, Setting Types

setting.name
The display name of the Setting

setting.saveName
The name used to save/retrive this Setting, no spaces

setting.invisible
Is true if the ui should hide this Setting


Changing/Reading values from Settings

You MUST read the type before doing anything,
to make sure you are not affecting/setting something you shouldn't

You can set the value normally, it will adapt depending on its
type to set the apropriate value to your new value


Value/Content of the setting

setting.value
Will give you the value of the Setting depending on its type
You can change the value too

setting.default
Will give you the default value of the Setting depending on its type
You can change it too if you want

Minimum/Maximum values

Not every type has a minimum/maximum value


Information can be found there

setting.min
The mainimum value of that Setting, you can change that value.

setting.max
The maximum value of that Setting, you can change that value.
