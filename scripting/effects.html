Effect Table

We will pretend that "effect" is an Effect


effect.duration
How long the effect lasts (in seconds) (number)

effect.level
The amplifier, ex: Regeneration 2 (number)

effect.id
The ID of the effect, a list of effect id can be found here (number)

effect.effectTimeText
The formated text of the time remaining (ex: 6:45) (string)

effect.name
The name that would be used in /effect (string)

effect.vname
The visual name (ex: Night Vision II) (string)
