Font Table

We will pretend that "font" is a Font Table


font.isMinecrafttia
Returns if the current font is (true)mojangles or (false)smooth (boolean)

font.height
The base height of the font (number)

font.wrap
For multiple lines, how much one lines will take (number)

font.width("string", (optional [number]scale))
Tells you how large the text is (number)
