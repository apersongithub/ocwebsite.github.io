Script Module

Everything in Module is in this too

We will pretend that "module" is a Script Module


module.movable
True if it has a position to change in the ui

module.scale
The scale (every position/size is multiplied by it automatically if movable) of the module

module.size
The size (as a table with a x and a y) of the module

module.pos
The position (as a table with a x and a y) of the module
