dimension Library

dimension.id()
Returns the id of the dimension the player is in (number)

dimension.name()
Returns you a name for the dimension the player is in (string)

dimension.time()
Normalised time of day in the dimension from 0.00000000 to 1.00000000 (number)

dimension.isRaining()
Returns if its raining or not (boolean)

dimension.sound([number]x, [number]y, [number]z, "Minecraft.Sound.Name")

Plays a minecraft sound at the coordinates a list can be found here


dimension.getBrightness([number]x,[number]y,[number]z)
Returns you the brightness level for the coordinates, [number]blockLight, [number]skyLight
local blockLight, skyLight = dimension.getBrightness(69,69,69)
--BlockLight is for things like torch, SkyLight is for the sun, does not adapt with time of day.

dimension.getBlock([number]x,[number]y,[number]z)

Returns the Block at these coordinates


dimension.getBiome([number]x,[number]y,[number]z)

Returns the Biome at these coordinates [info from mc not me]
