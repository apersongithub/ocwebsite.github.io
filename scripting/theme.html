Theme Table

here is what colors the ui contains, same order as theme editor

We will pretend that "theme" is a Theme Table




theme.back the background color, whats below everything

theme.moduleOutline the outline of the visual modules in the hud editor

theme.moduleOverlay the overlay on top of the visual modules in the hud editor

theme.darkbutton most buttons use "enabled" but some use darkbutton's color instead

theme.text color of the text on the ui

theme.largeArea color of the body of a window/pannel

theme.titlebar color of the titlebar

theme.disabled color of disabled stuff

theme.enabled color of enabled stuff, can be used as accent color

theme.blocked blocked content in that color
