Attributes Table

We will pretend that "attributes" is an Attribute Table

Holds a list of Attribute


attribute.size
The amount of attributes in the list (number)

attribute.name([string])

Returns the attribute Names can be found here (Attribute / nil if invalid)

attribute.id([number])

Returns the attribute IDs can be found here (Attribute / nil if invalid)

attribute.at([number])
Returns the attribute at the position, use for iteration only, not static/reliable (Attribute / nil if invalid)


You can Iterate trough the list like this

for i,attributes.size do
print(attributes.at(i).id .. ": " .. attributes.at(i).name)
end
