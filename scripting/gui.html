gui Library

gui.width()
Returns you the width of the screen (number)

gui.height()
Returns you the height of the screen (number)

gui.scale()
Returns you the minecraft guiscale (number)

gui.mouseGrabbed()
Returns you if the mouse is grabbed (if its not grabbed its most likely in an ui) (boolean)

gui.setGrab([boolean])
Sets the grabbed status of the mouse, can be used to make menus

gui.mousex()
Returns the x position of the mouse (number)

gui.mousey()
Returns the y position of the mouse (number)

gui.showScreen("ScreenName")
Opens the client screen with that name, returns [boolean](open or no)

gui.clickSound()
Plays the click sound, to make button clicks have a sound

gui.sound("Minecraft.Sound.Name")

Plays minecraft sound a list can be found here

gui.stopallsound()
Stops all the minecraft sounds currently playing


gui.theme()
Returns you the user's Theme (table)


gui.font()
Returns you the current Font (table)
