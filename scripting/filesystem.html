fs (FileSystem) Library


fs.open([string]Path, [string]openMode)

Gives you a File if it can open, open modes are 'w', 'r', 'a' a is like w but at the end of existing file

fs.exist([string]Path)
Returns true if the path points to an existing file/directory

fs.isdir([string]Path)
Returns true if the path is a directory

fs.mkdir([string]Path)
Makes the directories

fs.delete([string]Path)
Delete the file/directory at the path

fs.copy([string]PathFrom, [string]PathTo)
Copy the file (can't copy directories)

fs.delete([string]PathFrom, [string]PathTo)
Moves the file to an other location (can't copy directories)

fs.files([string]Path)
Returns a table with all file paths in that folder

fs.directories([string]Path)
Returns a table with all file paths in that folder

fs.stats([string]Path)
Returns a table with information on the file
The Stats table (s will represent a Stats table)
s.size -- the size in bytes of the file
s.readtime -- the last time the file was accessed (unix time)
s.writetime -- the last time the file was modified (unix time)
s.statustime -- the last time the file's status was changed (unix time)
