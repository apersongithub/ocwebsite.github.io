Inventory Table

We will pretend that "inventory" is an Inventory Table


inventory.size
The size of the inventory (number)

inventory.selected
The current slot that is selected (number)

inventory.armor()
Returns an InventoryArmor Table (see below) (table)

inventory.offhand()
Returns the offhand Item (table)

inventory.at([number])
Returns the item for the slot (Item / nil if no item for that slot)

if inventory.at(1) ~= nil then print("item in slot 1") end
