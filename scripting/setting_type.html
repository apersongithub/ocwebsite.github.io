Setting Types

Settings have different types, here is a list of them!

The type will change the behavior and you must check it to change/read Settings
you should not assume the type and it cannot be changed

1 Bool, true or false
2 Int, a number, ex: 1, 58, 456
3 Float, a number, ex: 0.5 69.420, 5.4

4 Keybind, a number for the key List of keys

5 Vec2, two Floats(3), it has a x and y value and you can do vec2 = {6.9,4.20}
6 Reserved for Vec3

7 ColorSetting, see Color Setting

8 FakeColor, same as ColorSetting but it only has r,g,b,a and doesn't have the extra fields

100 and more are not saved, they are "Cosmetic" settings for the ui

100 Air, puts space for the ui to split things, the height of the gap
101 Info, text that can be changed that to display information in the ui
102 Function, button that calls a function, you cannot change/call that function.


Types with Minimum / Maximum values

2 (or Int)
3 (or Float)
5 (or Vec2)
6 (or Vec3 (when implemented))
